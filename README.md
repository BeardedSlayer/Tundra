# [Aviation Cards Engine]

> A brief description: An OOP-based web application for managing and displaying military aircraft data.

## 🚀 Overview
This project was created to practice **Object-Oriented Programming (OOP)** in JavaScript and build a scalable system for data management. It focuses on clean code, modularity, and interactive user experience.

## 🛠 Tech Stack
* **Frontend:** HTML5, CSS3 (Flexbox/Grid), JavaScript (ES6+)
* **Logic:** OOP Classes, DOM Manipulation, Event Listeners
* **Tools:** Git/GitHub

## ✨ Key Features
* **Dynamic Card Generation:** Cards are created using JS classes to ensure data integrity.
* **Responsive Design:** Fully functional on mobile and desktop devices.

## 🧠 What I Learned
* How to structure complex logic using **JS Classes**.
* Managing the **DOM tree** efficiently without external libraries.
* Implementing **Clean Code** principles to make the project maintainable.

## 📦 Installation & Setup
1. Clone the repository: `git clone https://github.com/BeardedSlayer/Tundra`
2. Open `index.html` in your browser.
